# מעקב תכניות תכנון 🏙️

סוכן אוטומטי לבדיקה שבועית של סטטוס תכניות תכנון מ-[iplan.gov.il](https://www.iplan.gov.il), עם עדכון במייל.

---

## איך זה עובד

```
Google Sheets (מספרי תכניות)
        ↓
GitHub Actions (כל יום ראשון)
        ↓
iplan.gov.il API (בדיקת סטטוס)
        ↓
השוואה לסטטוסים משבוע שעבר
        ↓
מייל HTML עם שינויים + סיכום כולל
```

---

## הגדרה ראשונית (10 דקות)

### 1. Fork/Clone הריפו הזה

לחץ **Fork** (למעלה מימין) — זה יוצר עותק בחשבון שלך.

### 2. הכן Google Sheets

- הגיליון שלך **חייב** להיות פתוח לצפייה ציבורית:  
  `שיתוף → כל מי שיש לו את הקישור → צופה`
- שים לב לאיזו עמודה מספרי התכנית (A=0, B=1, C=2, ...)
- שים לב לאיזו עמודה שם/תיאור התכנית

### 3. הכן Gmail App Password

1. היכנס ל-[myaccount.google.com](https://myaccount.google.com)
2. `אבטחה → אימות דו-שלבי` (חייב להיות פעיל)
3. `אימות דו-שלבי → סיסמאות אפליקציה`
4. צור סיסמה חדשה עם שם "PlanningTracker"
5. שמור את ה-16 תווים שקיבלת

### 4. הגדר GitHub Secrets

ב-GitHub של הריפו שלך: `Settings → Secrets and variables → Actions → New repository secret`

| שם Secret | ערך | דוגמה |
|-----------|-----|--------|
| `SHEETS_URL` | קישור מלא לגיליון | `https://docs.google.com/spreadsheets/d/1GD-aH...` |
| `PLAN_COL` | מספר עמודת תכנית (0=A) | `2` (עמודה C) |
| `NAME_COL` | מספר עמודת שם תכנית | `1` (עמודה B), או `-1` אם אין |
| `HEADER_ROWS` | כמה שורות כותרת לדלג | `1` |
| `NOTIFY_EMAIL` | מייל לקבלת עדכונים | `urban.romema@gmail.com` |
| `GMAIL_USER` | Gmail שולח | `yourbot@gmail.com` |
| `GMAIL_APP_PASS` | App Password (ללא רווחים) | `abcdabcdabcdabcd` |
| `SEND_FULL_SUMMARY` | כלול טבלה מלאה במייל? | `true` |

### 5. הרצה ראשונה ידנית

`Actions → מעקב תכניות שבועי → Run workflow → Run workflow`

זה ישמור את הסטטוסים הראשוניים — ממנה והלאה כל שינוי יישלח אליך.

## תדירות בדיקה

ערוך את `.github/workflows/weekly-check.yml`, שורת `cron`:

```yaml
# כל יום ראשון 10:00 ישראל (07:00 UTC)
- cron: '0 7 * * 0'

# כל שבועיים (ראשון ראשון בחודש)
- cron: '0 7 1-7 * 0'

# פעמיים בשבוע (ראשון + רביעי)
- cron: '0 7 * * 0,3'

# יומי (לבדיקות תכופות)
- cron: '0 7 * * *'
```
